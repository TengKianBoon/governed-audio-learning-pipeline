# ADR-0001: Public Code, Private Raw Learning Vault

Date: 2026-06-20  
Status: accepted

## Context

The project should improve hireability and show enterprise AI engineering practice in public. Raw learning materials may contain copyrighted videos, paid course content, private notes, personal file names, or confidential context.

## Decision

Keep app code public-ready, keep raw audio and full transcripts private, and publish only curated sanitized learning outputs after human review.

## Why

This gives a strong build-in-public story without exposing source material or private data.

## Consequences

The project needs publishing gates and separate private/public output folders from day one.
